Teaching Loads

GET /teaching-loads View teaching assignments

Query Parameters (all optional):

school_year_id Filter by school year ID
school_level Filter by school level
program_code Filter by program code
year_level Filter by year level
section Filter by section
teacher_id Filter by teacher ID
course_id Filter by course ID
page Page number for pagination (default: 1)
limit Items per page (default: 10)
POST /teaching-loads Create new teaching load (Admin only)

Request Body (all required):

teacher_id Teacher ID
course_id Course ID
section Section name
year_level Year level
program_code Program code (optional)
school_level School level
school_year_id School year ID
semester Semester
role Teacher role
PUT /teaching-loads/:id Update teaching load (Admin only)
DELETE /teaching-loads/:id Deactivate teaching load (Admin only)

Grades

GET /grades Query student grades with filters

Query Parameters (all optional):

school_year_id Filter by school year ID
semester Filter by semester
program_code Filter by program code
year_level Filter by year level
section Filter by section
course_id Filter by course ID
teacher_id Filter by teacher ID (Admin only)

Example URLs:

/grades
/grades?school_year_id=2024&semester=1
/grades?program_code=BSIT&year_level=3§ion=A
/grades?course_id=IT101&teacher_id=1
POST /grades Update student grade

Request Body:

id Grade record ID (required)
grade_value New grade value (required)
